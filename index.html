Hvad bruger man semikolonner til i Java (og Processing)?

I Java (og også i Processing, da det bygger på Java) bruges semikolonner til at afslutte en kommando eller en linje med kode. Det fortæller kompilatoren, at en bestemt linje er slut, og den skal fortsætte med at analysere næste linje.

Hvad er datatyper, og hvilke datatyper kender du?

Datatyper angiver typen af data, en variabel eller en værdi kan indeholde. Nogle almindelige datatyper i Java inkluderer:

  • int: Heltal (f.eks. 5, -10)
  • double: Flydende kommatal (f.eks. 3.14, -0.5)
  • boolean: Sandt eller falsk (true eller false)
  • String: Tekst (f.eks. "Hej verden")
  • char: Enkelt tegn (f.eks. 'a', '$')
